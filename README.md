# WeatherApp - Monitorização Meteorológica

## Descrição
Aplicação Android desenvolvida para fornecer informações meteorológicas em tempo real. Este projeto serviu para consolidar conhecimentos sobre o ciclo de vida de uma Activity, gestão de permissões (GPS) e comunicação básica entre componentes Android.

## Funcionalidades
* Obtenção das coordenadas GPS atuais do dispositivo.
* Consulta de dados meteorológicos via API (Open-Meteo API).
* Interface dinâmica que se ajusta consoante o estado do tempo.
* Utilização de recursos XML para internacionalização e cores.

## Arquitetura e Tecnologias
* **IDE:** Android Studio
* **Linguagem:** Kotlin
* **Interface:** XML Views
* **Gestão de Dados:** Integração de recursos nativos Android

## Instalação
1. Abrir o projeto no Android Studio.
3. Deixar o projeto compilar automaticamente e correr a aplicação no emulador ou dispositivo físico (API 24+).
